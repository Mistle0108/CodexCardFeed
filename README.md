# CodexCardFeed

Codex 대화는 많은 토큰 비용을 들여 만들어지지만, 대부분은 그 순간의 작업이 끝나면 가치가 급격히 떨어지는 일회성 출력으로 남는다. 이 프로젝트는 그런 대화를 프로젝트, 스레드, 턴 단위로 수집하고 정리해, 버려지는 결과물이 아니라 다시 찾고 분류하고 관리할 수 있는 자료이자 재사용 가능한 지식 자산으로 전환하는 것을 목표로 한다.

CodexCardFeed는 Codex의 로컬 대화 데이터를 가져와 프로젝트, 채팅, 스레드, 턴 구조로 탐색할 수 있게 하고, 대화 내용을 카드 형태로 정리해 다시 활용할 수 있도록 돕는 로컬 Electron 도구다.
